# 수유시설 길찾기 서비스 MilkyWay 🍼
#### 모바일프로그래밍 프로젝트 결과물
<br/>

## 🖥️ 프로젝트 소개
`MilkyWay`는 모유수유 시설을 편리하게 찾아갈 수 있도록 도와주는 서비스입니다.
사용자 위치 기반으로 주변의 수유시설을 알려주고, 산모들의 불편함을 줄이는 것을 목적으로 합니다.
